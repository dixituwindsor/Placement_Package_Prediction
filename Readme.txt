Here are some steps to execute our project

1 first open Machine Learning folder
  - Dataset file is dataset which we have used to train and test our machine learning     models in classification and regression.
  - Classification_Jupyter_Notebook and Regression_Jupyter_Notebook are files of     our model implementation for classification and regression respectively.
  - Random_Forest_Classifier_Model.pkl and Random_Forest_Regression_Model.pkl files are     ML models in form of Pickle file.

2 PacementPrediction_Pycharm file is for api which we have created and deployed it on   heroku online plateform. So that we can use it in android application.

3 Placement zip file is androidstudio project file and it is our android application.

4 We have used Firestore of firebase for back-end or for store the data.
